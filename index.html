Abecedário!

Jogue enquanto pratica uma nova língua.



Regras:


Após sortear uma letra do alfabeto os jogadores deverão colocar uma palavra de cada categoria escolhida que comece com a letra sorteada no local indicado. Após um tempo, ou uma quantidade de participantes encerrar o turno, é feito a contagem de pontos. Cada categoria certa vale um ponto. O participante com mais pontos vence o turno, e o participante com mais turnos ganhos vence o jogo.


Modalidades:


  • Normal

    Nenhuma, ou uma quantidade de tempo longa para que cada turno possa ser concluído sem pressa.

  • Cronometrada

    É determinada uma quantidade de jogadores e de tempo para encerrar o turno. O turno acaba ou quando o tempo esgotar ou quando o número de jogadores determinado encerrar o turno. Ex.: 3 jogadores ou 3 minutos (nesse modo se 3 jogadores encerrarem seu turno ou se 3 minutos passarem o turno acaba).

  • Atenience, Espartana ou Romana

    É determinado um tempo para terminar a partida como um todo (sem divisões em rodadas), assim a partida acaba quando o tempo ou as letras possíveis do alfabeto acabarem. Nesse modo o jogador pode passar para uma nova letra quando quiser, e quando as letras esgotarem o jogador poderá revisar as letras anteriores em qualquer ordem, desde que ainda tenha tempo.
