Abecedário!

Jogue enquanto pratica uma nova língua.



Regras:


Após sortiar uma letra do alfabeto os jogadores deverão colocar uma paçavra de cada categoria escolhida que comece com a letra sortiada no local indicado. Após um tempo, ou uma quantidade de participantes encerrar o turno, é feito a contagem de pontos. Cada categoria certa vale um ponto. O participante com mais pontos vence o turno, e o participante com mais turnos ganhos vence o jogo.


Modalidades:


  • Normal

    Nenhuma, ou uma quantidade de tempo longa para que cada turno possa ser concluido sem pressa.

  • Cronometrada

    É determinada uma quantidade de jogadores e de tempo para encerrar o turno. Ex.: 3 jogadores ou 3 minutos (nesse modo se 3 jogadores encerrarem seu turno ou se 3 minutos passarem o turnu acaba).

  • Atenience, Espartana ou Romana

    É determinado um tempo para terminar a partida como um todo (sem divisões em rodadas), assim a partida acaba quando o tempo ou as letras possíveis do alfabeto acabarem.
